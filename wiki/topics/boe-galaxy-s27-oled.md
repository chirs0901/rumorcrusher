# 主题：BOE争夺三星Galaxy S27 OLED订单（2026年5月）

## 事件摘要
BOE Technology Group正以每片低于三星Display约$5的报价，争夺三星Galaxy S27基础款OLED面板供货权。三星MX部门已对BOE样品测试超一个月，BOE无重大技术障碍。

## 关键细节

| 项目 | 内容 |
|------|------|
| 目标机型 | Samsung Galaxy S27（基础款） |
| 价差 | BOE低于SDC约$5/片 |
| 测试状态 | 三星MX测试>1个月，无重大技术问题 |
| 月出货量 | 约130万台（×$5 = 月节省约650万美元） |
| S27+/Ultra | 继续使用SDC AMOLED 2X，不受影响 |
| 背景动因 | 中国手机品牌削减订单→BOE产能过剩→主动降价抢海外订单 |
