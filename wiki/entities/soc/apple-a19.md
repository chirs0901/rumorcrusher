# 苹果 A19 芯片

> 建档：2026-05-30 | AVeriTeC：Supported（苹果官方数据）| 置信度：0.97

## 基本信息

| 字段 | 值 |
|------|-----|
| 制造商 | 苹果（Apple Silicon） |
| 代工厂 | 台积电第三代3nm（N3E系列） |
| 发布时间 | 2025年9月 |
| 首发机型 | iPhone 17（标准版） |
| CPU | 6核（2×高性能核 + 4×高能效核） |
| GPU | 5核 |
| CPU性能提升 | 1.5倍于A15 Bionic（苹果官方） |
| GPU性能提升 | 2倍以上于A15 Bionic（苹果官方） |
| 神经网络引擎 | 支持苹果Apple Intelligence |

## 关键技术细节

- 台积电第三代3nm工艺（相较第一代3nm功耗效率进一步优化）
- 支持iPhone 17全系Apple Intelligence功能（含优先级处理本地AI推理）
- 与iPhone 17 Pro的A19 Pro区别：A19 Pro包含额外GPU核心和Pro级机器学习硬件
