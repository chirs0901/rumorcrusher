# 台积电 N2 (2nm) 工艺节点

## 基本信息

| 属性 | 值 |
|------|-----|
| 制程代号 | N2 / N2P |
| 晶体管密度 | 比N3E提升~15% |
| 性能提升（vs N3E） | +10%~15%（同功耗） |
| 功耗降低（vs N3E） | -25%~30% |
| 晶体管架构 | GAAFET（Gate-All-Around） |
| 量产时间 | N2: Q4 2025；N2P: H2 2026 |
| 初期月产能 | ~3.5万片（2026年1月）→目标14万片（2026年底） |
| 代工报价 | ~$30,000/晶圆 |
| 试产良率 | >70% |

## 关键客户
- 苹果（A20 Pro，独占初期约50%产能）
- 联发科（天玑9600，首款2nm手机芯片）
- 高通（Snapdragon 8 Elite Gen 6系列）
- AMD、博通、英特尔
