# 硅碳电池（Silicon-Carbon Battery）

## 基本信息

| 属性 | 值 |
|------|-----|
| 技术原理 | 用硅（部分或全部）替代石墨阳极，提升能量密度 |
| 主要优势 | 同体积下容量提升20~40%；支持超薄大容量设计 |
| 主要挑战 | 膨胀率高、长期可靠性、制造良率、监管认证 |
| 中国品牌采用 | ✅ OnePlus 15 / iQOO 15 Ultra / 荣耀Magic Win（2026年已普及） |
| 三星 | 2026年2月承认评估中，最早2026年底或2027年 |
| 苹果 | 尚未采用 |

|------|---------|------|
