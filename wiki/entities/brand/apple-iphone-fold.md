# 苹果 折叠iPhone（iPhone Fold / iPhone Ultra）

## 基本信息

| 属性 | 值 |
|------|-----|
| 外屏尺寸 | 5.5英寸 |
| 内屏尺寸 | 7.8英寸 |
| 屏幕供应商 | 三星Display（700~800万片折叠面板） |
| 铰链供应商 | Amphenol（美国，主供） |
| 铰链材料 | 液态金属铰链 + 结构玻璃中框 |
| 折痕解决方案 | 与SDC合作重构面板层叠结构（供应链声称，未实机验证） |
| 预计量产开始 | 2026年8月（从原计划6月延迟） |
| 预计发布 | 2026年秋季（与iPhone 18系列同步） |
| 预计起售价 | $2,000~$2,500 |
| 主要延迟原因 | 铰链在高频开合耐久测试中持续未达质控标准 |

## 供应链关键节点
- SDC作为折叠OLED独家供应商（苹果折叠屏），约700~800万片/年
- Amphenol最终胜出铰链竞标（此前有韩国和中国厂商参与）
- Lens Technology为超薄玻璃（UTG）主要候选供应商
