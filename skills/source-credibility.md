# 信源可信度档案 / Source Credibility Ledger

> 每个信源的可信度会随每日质检结果动态调整。这里记录历史轨迹和重要事件。

## 评分维度（每次质检后由Agent填写）

- **factual_accuracy**: 事实准确性（0-1）
- **completeness**: 信息完整度（含上下文）
- **transparency**: 信源透明度（明确标注利益关系、广告）
- **timeliness**: 时效性
- **clickbait_index**: 标题党指数（数值越低越好）
