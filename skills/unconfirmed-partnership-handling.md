# Unconfirmed Partnership Handling
## 官方未确认技术合作声称处理规则
- 无论技术逻辑多可信，缺少官方确认时最高只能标ConflictingEvidence
- summary中必须注明"官方未确认"
- 若涉及跨国合作（如美中合规），必须提示合规风险
- 典型案例：Qualcomm+CXMT 3D DRAM NPU（2026-06-01，ConflictingEvidence）

## 追加条款（2026-06-02）
- 若多家媒体报道同一未确认合作，但关键细节（如供应商选择）相互冲突，不因报道数量多而升级为Supported
- 典型案例：OpenAI+Qualcomm/MediaTek（4月报道称双供应商；5月报道称仅MediaTek）
