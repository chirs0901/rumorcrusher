# GPU性能"第一"声明核查规则

## 适用场景
芯片厂商在发布前或发布时声称GPU性能为"行业第一"

## 核查规则
1. **单一来源自我声明** → CE + logic_flag=true（单一来源谬误）
2. **官方参数+工艺支持** → CE（等待第三方基准）
3. **第三方基准测试（AnTuTu GPU/Geekbench GPU/3DMark）确认** → 可升级Supported
4. **与前代比较的相对提升** → 通常Supported（厂商内部测试可信）

## 注意事项
- GPU性能受驱动优化影响，早期驱动往往低估最终水平
- 不同基准测试结果可能差异显著（综合分 vs 图形分）

## 已知案例
- 天玑9600 GPU第一声明：单源/发布前 → CE + logic_flag
- Hexagon NPU 37%提升（与前代比）：官方数字 → Supported
