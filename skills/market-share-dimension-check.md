# 市场份额数据核查规则

## 核查维度
声称市场份额时必须明确：
1. **地理范围**：中国市场 / 全球市场 / 特定地区
2. **统计口径**：出货量(shipment) / 销售额(revenue) / 激活量
3. **时间范围**：季度 / 半年 / 年度
4. **品类范围**：折叠屏 / 旗舰 / 整体手机市场

## 核查规则
- 单维度声明且来源可信 → Supported
- 维度混淆或来源为厂商自称 → CE
- 无具体数字只有"领先" → NEE

## 已知案例
- 华为71.8%折叠屏：中国市场/出货量/2025全年 ✅ Supported
- "iPhone全球第一"：需区分出货量(通常三星领先)vs高端段收入(苹果领先)
