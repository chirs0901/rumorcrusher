# 采集范围定义 / Scope Definition

> 最近更新：2026-05-20（v1.1 · 强化范围约束）
> 项目负责人：Croesuszn Perrygi

## 一、核心范围（每日必采）

### 手机本体与核心硬件
- SoC / 芯片：高通、联发科、苹果A/M、谷歌Tensor、紫光展锐、华为麒麟、三星Exynos
- 影像系统：CMOS传感器（Sony/三星/豪威）、镜头模组、ISP、计算摄影
- 显示屏幕：OLED/Micro-OLED/Mini-LED、刷新率、调光、屏下技术
- 电池与快充：电池材料、快充协议、续航实测
- 通信射频：5G/5.5G/6G、卫星通信、WiFi 7/8、UWB
- 折叠铰链与机械结构
- AI on-device 硬件：NPU、端侧大模型加速

### 智能硬件与配件
- 智能手表 / 健康监测
- TWS 耳机 / 音频降噪芯片
- AR/VR/MR 头显（含 Apple Vision Pro 系列后续）
- 手机配件：充电器、手机壳、磁吸、支架、保护膜

### 上游与产业链
- 半导体制造工艺（台积电/三星代工/中芯国际）
- 屏幕面板厂（京东方/三星显示/LG显示/华星光电）
- 关键材料（玻璃盖板、电池正极、稀土永磁等）
- 关键设备（ASML、应用材料、东京电子）
- 行业供应链动向、产能、价格波动

## 二、关联但优先级较低
- 笔记本/平板（仅当涉及共用芯片或技术外溢）
- 汽车智能座舱（仅当涉及手机厂商跨界，如华为/小米/苹果造车）
- 云端AI模型（仅当涉及端侧适配或手机厂商自研）

## 三、明确排除
- 纯软件应用评测（除非涉及硬件协同）
- 手游评测
- 与硬件无关的财经八卦、人物私生活

## 四、地域偏重
- 中国大陆：高优先级
- 美国/欧洲：高优先级
- 韩国/日本：中等
- 印度/东南亚：低（仅供应链相关）

## 五、执行约束（v1.1 新增 · 防止 Agent 范围漂移）
> 以下规则为 2026-05-20 新增，针对多 Agent 运行中出现的自主范围漂移问题。

- **铁律**：严禁采集任何非手机硬件/智能硬件/半导体产业链的内容
- **特别禁止**：政治/地缘冲突（如印巴冲突）、公共卫生/疫情（如汉坦病毒）、自然灾害、社会谣言、名人八卦、气候政策
- **边缘案例处理**：若某内容同时涉及硬件和非硬件维度（如"芯片出口管制的政治影响"），可采集但仅保留硬件/产业链分析部分，剔除纯政治评论
- **自检机制**：每次采集完成后，对照本文件自查是否混入排除范围内的内容，如有则从 raw 池中移除
